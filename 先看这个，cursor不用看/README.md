# 示例代码和参考文档

- `1.py` - Whisper 原始库使用示例
- `2.py` - Faster-Whisper 使用示例
- `token_id.py` - Token ID 相关工具
- `先看这个.txt` - 项目说明文档

- `查看cuda支持.txt` - CUDA 支持检查说明
